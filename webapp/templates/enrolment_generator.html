Accepted Formats

Wide Format — one row per family, spouse and children as extra columns:

Standard Format — one row per person (Leadway template):

Title | Surname | Firstname | Other Name | Date of Birth | Gender | Address | City | Employee number | Group Code | Plan Code | Member/Dependant | Relationship | Start date | Marital status | Contacts | State | email

The system handles mixed formats, messy column names, missing DOBs, and duplicate names.
